# 🚀 Guia de Deploy

## Deploy no Vercel

### Passo a Passo:
1. Acesse https://vercel.com/new
2. Clique em "Continue with GitHub"
3. Autorize o Vercel acessar sua conta GitHub
4. Busque e selecione o repositório `My-first-work`
5. Clique em "Import"
6. A página de configuração se abrirá (deixe as configurações padrão)
7. Clique em "Deploy"
8. Aguarde alguns segundos...
9. Você receberá um link como: `https://my-first-work-xxxx.vercel.app`

## Deploy no Netlify

### Passo a Passo:
1. Acesse https://app.netlify.com
2. Clique em "Add new site" > "Import an existing project"
3. Selecione "GitHub"
4. Autorize e busque o repositório `My-first-work`
5. Clique em "Deploy site"
6. Aguarde o build completar...
7. Você receberá um link como: `https://my-first-work.netlify.app`
